# Lab-9-Numerical-Integration
To numerically obtain the integration of any given function

This repository includes MATLAB implementations of numerical techniques employed to estimate derivatives and definite integrals. The primary aim of this project is to comprehend the application of these methods when analytical solutions are challenging and to evaluate their precision against exact outcomes.

The study centers on two main aspects: numerical differentiation and numerical integration. In differentiation, the forward difference technique is applied once uneven data is transformed into uniform intervals. The findings indicate that the approximation error is greater at the initial point and decreases for subsequent values because of the method's characteristics.

For integration, the provided function is assessed using the trapezoidal rule and Simpson’s rules. The trapezoidal method exhibits greater error with fewer intervals, but its precision enhances as the interval count rises. Simpson’s methods yield superior outcomes since they more accurately approximate the curve, with Simpson’s rules producing values that are nearer to the analytical solution.

In summary, this project emphasizes that numerical techniques are useful for addressing real-world issues, yet their precision relies on the chosen method and the size of the steps. It likewise shows that higher-order methods and longer intervals result in improved approximations, although some error is consistently present
